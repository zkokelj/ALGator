This presenter presents Dinics and EdmonsKarp algorithm on TestSet4.
TestSet4 test cases have the same number of nodes, but significantly less edges then test cases in TestSet2. Time complexities for this two algorithms are:
Edmonds: O(VE^2)
Dinics: O(EV^2)
Compared to previous graph we can see that EdmonsKarp algorithm does well with graphs that have small(er) number of edges.
