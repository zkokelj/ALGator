A test case used in maximum flow problem is a weighted directed graph/network, where weights need to be integer values. Network has to have at least one node that has no ingoing edges (called source) and at least one node that has no outgoing edges (called sink). Nodes are labeled with integer values from 0 to n - 1.

For the Java programs the test case is declared with
int n;
int[][] g = new int[n][n];
int s;
int t;
where n is the number of nodes in the graph, s is the source node and t is the sink node.
