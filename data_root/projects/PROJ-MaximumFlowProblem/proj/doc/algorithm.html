Maximum flow problem involve finding maximum feasible flow through network/graph with single source and single sink.
Vertices are connected with edges. Each edge has a capacity. Main goal is to figure out maximum possible flow. We have to take into account limitations of edge capacities and that in each vertex input and output flows should be equal.

Input: Graph g - (with list of connected neighbours) n - number of verices s - index of source vertex t - index of sink vertex
Output: flow - maximum possible flow from source to sink graph g - graph of flows that produce maximum flow from source to sink
Test of correctness:     If flow is equal to expected maximal flow In returneg graph respects all limitaions of input graph
Implementation: Dinics algorithm EdmonsKarp algorithm FordFulkerson algorithm
